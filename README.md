# IoT weather station

## Components

- Esp8266 PRO d1 mini module.
- Current Sensor.
- 330 ohm Resistence.
- 2x 10 kohm Resistance

## Firmware Struct

Using esp IDF, Espressif IoT Development Network, the following modules were created using the Esp IDF as a SDK:

- TCP/IP Module.
- MQTT Client Module.
- Current Read Module.
